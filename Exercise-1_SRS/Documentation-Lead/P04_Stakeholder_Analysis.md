| Stakeholder      | Role       | Interest                                     | Impact                   |
| ---------------- | ---------- | -------------------------------------------- | ------------------------ |
| Home Gardeners   | End Users  | Accurate plant advice, easy-to-use interface | High - System adoption   |
| Development Team | Builders   | Learn MLOps, React, Flask; deliver quality   | High - Project execution |
| Project Mentor   | Supervisor | Technical depth, timely completion           | High - Academic grading  |
