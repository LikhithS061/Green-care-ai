## Glossary Development

| Term            | Definition                                                                       |
| --------------- | -------------------------------------------------------------------------------- |
| **API**         | Application Programming Interface - protocols for building software applications |
| **CNN**         | Convolutional Neural Network - deep learning algorithm for image recognition     |
| **Flask**       | Lightweight Python web framework for building APIs                               |
| **JWT**         | JSON Web Token - secure method for authentication                                |
| **MLOps**       | Machine Learning Operations - practices for deploying ML models                  |
| **MongoDB**     | NoSQL document database for flexible data storage                                |
| **NPK**         | Nitrogen-Phosphorus-Potassium - primary plant nutrients                          |
| **pH**          | Measure of acidity/alkalinity (scale 0-14)                                       |
| **PWA**         | Progressive Web App - web app with mobile-like experience                        |
| **React.js**    | JavaScript library for building user interfaces                                  |
| **SHAP**        | SHapley Additive exPlanations - method to explain ML predictions                 |
| **Transformer** | Deep learning architecture for sequence processing                               |
| **YOLO**        | You Only Look Once - real-time object detection algorithm                        |
| **YOLOv8**      | Version 8 of the YOLO object detection model                                     |
