## Functional Requirements

| ID    | Requirement          | Description                                                        | Priority |
| ----- | -------------------- | ------------------------------------------------------------------ | -------- |
| FR-01 | User Registration    | System shall allow users to create a profile with email/password   | High     |
| FR-02 | User Login           | System shall authenticate users with valid credentials             | High     |
| FR-03 | Environmental Input  | System shall accept N, P, K, pH, and Temperature values            | High     |
| FR-04 | Input Validation     | System shall validate input ranges before processing               | High     |
| FR-05 | Plant Recommendation | System shall display top 3 plant matches with confidence scores    | High     |
| FR-06 | Image Upload         | System shall allow image upload (JPG/PNG, max 5MB)                 | High     |
| FR-07 | Disease Diagnosis    | System shall identify plant diseases from uploaded images          | High     |
| FR-08 | Remedy Display       | System shall display treatment suggestions for identified diseases | High     |
| FR-09 | Care Instructions    | System shall provide care plans for recommended plants             | Medium   |
| FR-10 | Profile Management   | System shall allow users to update their profile information       | Medium   |
