# Glossary

## Greencare AI Project

| Term             | Definition                                                                                        |
| ---------------- | ------------------------------------------------------------------------------------------------- |
| **API**          | Application Programming Interface - a set of protocols for building software applications         |
| **CNN**          | Convolutional Neural Network - a deep learning algorithm used for image recognition               |
| **Flask**        | A lightweight Python web framework used for building APIs                                         |
| **MongoDB**      | A NoSQL document database used for storing user and plant data                                    |
| **NPK**          | Nitrogen-Phosphorus-Potassium - the three primary nutrients for plant growth                      |
| **pH**           | A measure of acidity/alkalinity of soil (scale 0-14)                                              |
| **Potting Mix**  | Soil-less medium used for container gardening                                                     |
| **React.js**     | A JavaScript library for building user interfaces                                                 |
| **SHAP**         | SHapley Additive exPlanations - a method to explain machine learning predictions                  |
| **SRS**          | Software Requirements Specification - a document describing software requirements                 |
| **Transformer**  | A deep learning architecture used for sequence-to-sequence tasks                                  |
| **YOLO**         | You Only Look Once - a real-time object detection algorithm                                       |
| **YOLOv8**       | Version 8 of the YOLO object detection model                                                      |
| **Microservice** | An architectural style that structures an application as a collection of loosely coupled services |
| **MLOps**        | Machine Learning Operations - practices for deploying and maintaining ML models in production     |
