## Scope Definition – In-Scope Features

The following features are included in the project scope:

1. **User Profiling** – Users can create profiles with soil type and location inputs
2. **Environmental Data Input** – Accept N, P, K, pH, and Temperature values
3. **Plant Recommendation Engine** – Transformer-based AI suggests top 3 plants
4. **Plant Disease Diagnosis** – YOLOv8-based image recognition identifies diseases
5. **Remedy Suggestions** – Provide treatment recommendations for detected diseases
6. **Web Interface** – React.js responsive frontend
7. **Backend API** – Flask-based REST API
8. **Basic Explainable AI** – SHAP analysis for recommendation transparency
