## Project Title

Greencare AI: Personalized Plant Recommendation and Gardening Assistant

## Project Description

This project aims to design and document an intelligent AI-powered application
that provides personalized plant recommendations based on environmental parameters
(NPK, pH, location) and real-time disease diagnosis via image recognition,
thereby helping gardeners make informed decisions and reduce plant mortality.
