## Class Identification

### Domain Classes

| Class           | Description                                   |
| --------------- | --------------------------------------------- |
| User            | System user/gardener with profile information |
| SoilProfile     | Environmental data (NPK, pH, temperature)     |
| Plant           | Plant information and care instructions       |
| Disease         | Disease details and symptoms                  |
| Recommendation  | AI-generated plant suggestions                |
| DiagnosisReport | Disease detection results                     |

### Service Classes

| Class                 | Description                           |
| --------------------- | ------------------------------------- |
| AuthService           | User authentication and authorization |
| RecommendationService | Plant recommendation logic            |
| DiagnosisService      | Disease detection processing          |
| StorageService        | Image upload and storage              |
